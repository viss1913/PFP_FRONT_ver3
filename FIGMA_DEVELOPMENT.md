## Следующие шаги

### 1. Проанализировать структуру дизайна

После запуска приложения (`npm run dev`) и открытия страницы результатов:

1. Откройте консоль браузера (F12)
2. Выполните команду: `viewFigmaStructure()`
3. Изучите структуру дизайна в консоли

Или структура будет отображена на странице в разделе "Структура дизайна (для анализа)".

### 2. Создать компоненты на основе дизайна

Изучите структуру дизайна и создайте React компоненты, которые точно повторяют:

- **Макет и расположение элементов**
- **Стили (цвета, шрифты, отступы)**
- **Компоненты (карточки, кнопки, списки)**

### 3. Подстановка данных

Используйте данные из `calculationData` для заполнения:

```typescript
// Пример структуры данных из расчета
{
  client: {
    first_name: string;
    last_name: string;
    // ...
  },
  goals: Array<{
    name: string;
    target_amount: number;
    // ...
  }>,
  plans: Array<{
    goal_name: string;
    monthly_payment: number;
    // ...
  }>
}
```

### 4. Обновить компонент ResultDesignTemplate

В файле `src/components/ResultDesignView.tsx` обновите компонент `ResultDesignTemplate`, чтобы он соответствовал дизайну из Figma.

## Полезные функции

- `viewFigmaStructure()` - в консоли браузера для анализа структуры
- `window.__FIGMA_STRUCTURE__` - полная структура дизайна для анализа
